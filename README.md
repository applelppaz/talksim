# TalkSim — 多言語会話生成・語彙入れ替え・発音練習アプリ

英語・中国語・スペイン語・フランス語の会話文をシチュエーションに合わせて生成し、定型文の中の単語を入れ替えながら、リスニング・スピーキング・語彙力を高めるための Gemini API ベースの Web アプリです。マイク録音と AI による発音／応答内容の評価にも対応しています。

## 使い方

1. **言語を選ぶ** — English / 中文 / Español / Français から選択
2. **シチュエーションを日本語で入力** — 例：「カフェで店員にラテを注文する。ミルクの種類を聞かれる想定。」
3. **「会話を生成」** をクリック — その場面で使える 2 人の登場人物による会話文を AI が生成
4. **音声で聞く** — 生成直後に全文が自動再生されます。各行に 🔊 ボタン、上部に「すべて再生」ボタンも
5. **語彙を入れ替える** — 色付きのボタン部分が入れ替え可能な単語/フレーズです。クリックすると 8〜12 個の候補がポップアップで表示され、選んだ表現に差し替わります
6. **役を選んで練習する** — 「あなたの役」から登場人物を選ぶと、自分の役のセリフに 🎤 録音ボタンが出ます。押して音読すると、AI が発音／応答内容を評価し、間違いを日本語で指摘します
7. **ロールプレイ練習** — 「🎙 ロールプレイ練習を開始」を押すと、相手のセリフは AI が読み上げ、自分のセリフでは録音を待つガイド付き練習に入ります

## 主な機能

- **会話文の自動生成**：シチュエーションに沿った 6〜10 往復の自然な会話を作成
- **定型文の語彙入れ替え**：1 セリフ内の重要箇所に複数のスワップポイントを設定し、それぞれに豊富な候補語を用意
- **自動音声再生**：生成直後に全体を順番に読み上げ（行ごとの個別再生も可）
- **話者切り替え／発音評価**：会話のどちらの役を担当するかを切り替えられ、自分のセリフをマイクで録音すると AI が発音や応答の正確さを採点し、改善ポイントを指摘
- **ロールプレイ練習モード**：相手役は AI が自動で発音、自分の役では録音を待つ対話型の練習フロー
- **TTS の選択**：ブラウザ標準 TTS / Gemini 高品質 TTS から選択可
- **API キーのローテーション**：Gemini API キーを最大 3 つ登録でき、無料枠の上限到達時に自動で他キーへ切替

## セットアップ

```bash
npm install
npm run dev
```

`http://localhost:5173` を開きます。

### API キーの取得

1. [Google AI Studio](https://aistudio.google.com/apikey) で無料の Gemini API キーを作成
2. アプリの「設定」ページに最大 3 つまで貼り付けて保存
3. キーはブラウザの localStorage にのみ保存され、外部送信されません

> 以前のバージョンで登録した API キーはそのまま使えます（保存場所は同じ `talksim:settings` です）。

## ビルド & デプロイ

```bash
npm run build      # dist/ を生成
npm run preview    # ローカルでビルド結果を確認
```

`netlify.toml` を同梱しています。Netlify にリポジトリを接続すれば、追加設定なしでデプロイ可能です（Vercel 等でも同様に動作します）。

## 技術スタック

- Vite + React 18 + TypeScript
- Tailwind CSS
- React Router v6
- `@google/genai`（公式 Gemini SDK）
- Web Speech API（標準音声合成・音声認識）

## 動作環境

- 発音評価（マイク入力）は Web Speech API の音声認識を利用しているため、**Chrome / Edge を推奨**します。Firefox / Safari では録音ボタンが無効になります。
- マイク使用時はブラウザの許可が必要です。
- 中国語の音声合成は OS 依存です。品質が気になる場合は設定で Gemini TTS に切り替えてください。

## データの取り扱い

すべてのデータ（API キー・生成済み会話の履歴）は、お使いのブラウザの `localStorage` にのみ保存されます。サーバには一切送信されません。
